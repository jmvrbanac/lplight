Welcome to the LPLight Library's documentation!
================================================================

LPLight is designed to be a lightweight REST client library for Launchpad.

.. note::

	This project has no affiliation with Launchpad or Canonical Ltd.


Library Goals:
	Simple
	Minimal Dependencies

Installation
---------------
The Library is available on PyPI.

.. code::

	pip install lplight
